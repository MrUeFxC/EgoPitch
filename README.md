# 项目启动方案：世界杯球星 AI 模拟访谈系统 (World Cup Star-Agent Podcast Simulator)

本项目旨在打造一款由 **AI Agent** 驱动的垂直类互动软件，通过模拟顶级球星（如 C罗、梅西、姆巴佩）参加深度访谈（参考摩根采访 C罗），制造出具有“足坛地震”级影响力的爆炸性内容。

---

## 一、 核心产品逻辑：认知引擎 (Cognitive Engine)

不同于传统的对话机器人，本系统的核心是基于 **“人格分层”** 与 **“渐进式意识读取”** 的思维逻辑。

### 1. “球星即 Skill” 的定义
球星不再是一段死板的 Prompt，而被封装为一个**可插拔的逻辑组件 (Logic Skill)**。
* **配置驱动：** 使用轻量化的 JSON 配置文件代替沉重的向量数据库。
* **渐进读取：** 初始仅加载“公共人格”，随着访谈刺激（提问、评论），动态激活深层人格逻辑碎片。
* **多重人格：** 模拟内心冲突（例如：职业素养 vs. 极度自尊），通过权重计算最终输出。

---

## 二、 架构设计：三层人格演化模型

系统将球星的思维逻辑拆解为三个渐进激活的层级（Layers），每个层级是一个独立的 Skill 片段。

| 层级 | 人格状态 | 逻辑行为 | 激活条件 |
| :--- | :--- | :--- | :--- |
| **Layer 1** | **公共人格 (Public)** | 职业化、官方词令、回避争议。 | 默认开启 (访谈开场) |
| **Layer 2** | **深层自我 (Ego)** | 展现极强自尊、开始引用个人荣誉、语气变生硬。 | 提及敏感词（如“替补”、“梅西”）次数达标 |
| **Layer 3** | **核爆模式 (Nuke)** | 彻底撕破脸、爆料更衣室内幕、点名批评。 | 怒气值 > 80% + 主持人诱导提问 |

### 核心公式：怒气值计算
$$Current\_Anger = (Trigger\_Words \times 10) + (Negative\_Social\_Sentiment \times 5)$$

---

## 三、 关键功能模块

### 1. 诱导式主持人 Agent (The Instigator)
* **任务：** 扫描球星的“痛点”，不断通过追问测试球星的反应。
* **逻辑：** 如果发现提到某个话题时球星怒气上升，主持人 Agent 会持续深挖该话题。

### 2. 实时舆论注入 Skill (Social Media Pulse)
* **功能：** 模拟 X (Twitter)、TikTok 的真实评论流。
* **互动：** 评论不仅是背景板，它会作为“外部刺激”输入给球星 Agent。
* **爆炸点：** 允许球星在访谈中直接回怼某条“高赞黑粉评论”，打破第四面墙。

### 3. 多模态输出链路
* **文本流：** 实时生成的对话内容。
* **音频流：** 接入 ElevenLabs API，克隆球星标志性的语调、重音和呼吸声。
* **视觉流：** 自动匹配球星在场上的落寞/愤怒静态图或 AI 生成短视频片段。

---

## 四、 技术栈方案 (MVP 轻量版)

* **推理大脑：** `Claude 3.5 Sonnet` 或 `GPT-4o` (具备极强的角色扮演能力)。
* **Agent 框架：** `LangGraph` (用于管理“平和”到“爆炸”的状态转换)。
* **存储与逻辑管理：** `State-Driven JSON` (存放球星人格碎片与逻辑开关)。
* **前端：** `Next.js` (实现类似直播间的实时滚动评论和对话效果)。
* **语音：** `ElevenLabs API` (音色克隆)。

## 五、 实施计划 (Roadmap)

### 第一阶段：原型构建 (Week 1-2)
* 编写 C罗 (CR7) 的 JSON 逻辑配置文件。
* 实现“关键词监听器”，完成 Layer 1 到 Layer 2 的平滑切换。
* 在控制台模拟“主持人-球星”对话，调优爆料概率。

### 第二阶段：舆论集成 (Week 3)
* 开发“虚假评论生成器”，按不同流派（人迷、黑粉、理客中）生成评论流。
* 接入情绪权重逻辑，让评论能够真实影响球星的回答。

### 第三阶段：多模态与分发 (Week 4)
* 接入 ElevenLabs 语音合成。
* 开发“一键生成金句海报”功能，适配社交平台分发。
* 进行世界杯场景测试（如：模拟葡萄牙出局后的“爆炸采访”）。

---

## 六、 项目愿景

这款软件不仅是一个聊天工具，它是一个**虚拟的足球娱乐圈生成器**。在世界杯期间，它将通过“预言式爆料”和“情绪化对线”，成为社交媒体上的流量收割机。

> **核心原则：** 逻辑要简单，效果要爆炸。不求 AI 懂足球战术，但求 AI 懂球星脾气。
